# Food Shelf

# Development
## Prerequisites
* Ruby 2.6.x
* Rails 6.x

### Windows
You may want to see https://gorails.com/setup/windows/10

## Setup
Assuming you have Ruby and Rails installed already

    gem install bundler
    bundle install
    bin/rake db:migrate

## Run the application

    bin/rails server

Browse to http://localhost:3000 to access the application

## Testing

    bin/rails test
